# BiblioCat API

BiblioCat API is a backend API for a (non-existent) library application.

Users can search for books that are available in the library, borrow books, review them and log their reading history.

Note that some features haven't been added yet.  
For the list of requirements, see: [Requirements](docs/requirements.md)
