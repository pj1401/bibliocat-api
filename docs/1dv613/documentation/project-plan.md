## Scope

*What is this project? What are you NOT building (to keep it realistic)?*

**In scope:**
- Server-side kod.
- Backend system.
- API
- Sessions eller JWT
- Kommunikation med externa API:er, tex. Google Books API.

**Out of scope:**
- SSR, vyer. Men det ska vara möjligt att vidareutveckla projektet, så att vyer kan läggas till.
- SPA: Fokus är på server kod.
- Microservices: Jag tror inte projektet är tillräckligt stort för att dela upp det i microservices.
